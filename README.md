# Git Secrets Scanner

Git repository secrets scanner.

Pemindai secret repository Git.


## Description

Searches git repositories for hardcoded secrets, API keys, and passwords in commit history.

## Deskripsi

Mencari repository git untuk secret hardcoded, kunci API, dan password dalam riwayat commit.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Scan: python git_secrets.py /path/to/repo
- Demo: python git_secrets.py --demo

## Penggunaan

- Pindai: python git_secrets.py /path/to/repo
- Demo: python git_secrets.py --demo
